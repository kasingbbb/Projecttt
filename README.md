# Projecttt - AI 项目仓库

## 项目概述
这是一个包含多个研究项目的 AI 项目仓库，主要用于新人群需求探索与研究、产品需求挖掘和验证框架的开发。

## 当前包含项目

### Project 一：新项目研究
这是一个系统化的新人群需求探索与研究项目，旨在建立完整的产品需求挖掘和验证框架。

**项目启动时间**: 2026年2月14日  
**项目负责人**: AI产品经理  
**项目目标**:
1. 建立系统化的新人群识别与评估框架
2. 开发科学的需求挖掘与验证方法论
3. 创建可复用的研究工具和模板
4. 产出高质量的用户洞察和产品建议

**核心方法论框架**:
1. **人群选择框架（4P模型）**: Potential, Payment, Pain, Plasticity
2. **价值判断标准（5C模型）**: Confirmation, Concentration, Cost, Competition, Capability
3. **需求挖掘四层漏斗**: 现象收集→需求提炼→需求验证→优先级排序
4. **MI/CI双视角研究**: 市场洞察与客户洞察结合
